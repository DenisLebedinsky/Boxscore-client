## Boxscore client 

This is client for server-side app 
https://github.com/DenisLebedinsky/Boxscore


### Installing

```
npm  install
```


## Running the tests

```
npm run test
```

### Running prodaction

```
npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
